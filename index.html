Git Commands

git init: for initialising

git status: for checking the status

git add "." or file name: for adding files to repository whereas . represents all files or we can mention file name

git commit -m "message": for commiting

git log --oneline: for checking the logs of commit(after "--"" no space is required)

git log: for detailed commit details

git checkout "id" "file name": will give the file with that commit

git reset HEAD "file name": will revert the staging of added files

git checkout -- "fiel name": will restore back to original file (after "--" space is required)
